Code in the Dark Tunis 2k19

In Tunisia for the first time. Big success. C u @ next. Code in the Dark is a front-end (HTML, CSS) competition, originating from Tictail, where each contestant compete to implement a website design given only a screenshot. The catch is that no previews of the results are allowed during the implementation, and no measuring tools can be used. The winner is decided by the audience. During the competition, the contestants have to use our custom editor for Code in the Dark. The competition will take place in several rounds.
